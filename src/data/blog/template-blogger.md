Hallo semua pada kesempatan kali ini saya akan share pengalaman pribadi dimana membeli template blogger di kangrian, jadi koleksi saya template kang rian ada 3 biji yaitu Media ID Blogger, Media ID Wordpress dan ini SimPerson ini. 

Karena ini blog pribadi dan isinya unek - unek jadi tidak lah perlu template banyak fitur dan tetek bengek lainnya.

Ditambah lagi di lapak kangrian ini sendiri ada promo sampai akhir tahun, tidaklah heran dari dulu kalau membeli template untuk harga nya sangat wow akan tetapi sebanding dengan apa yang di dapat. 

## 1. Harga Template
Seperti yang saya katakan kalau di kangrian ini harga template nya sangat wow, harga template yang saya pakai ini sendiri aja 100.000 rupiah karena ada promo menjadi 50.000 rupiah. 

Untuk membelinya template yang saya pakai langsung aja klik link ini

## 2. Tampilan Template
Karena template SimPerson ini digunakan untuk blog pribadi jadi tidak lah neko - neko ya gaes, cukup ada tombol share, navigasi menu, serta ada search dan label. 

Tapi untuk template ini sendiri hapir sesui karakter template yang saya idam - idamkan dan itupun gak terlalu banyak fitur yang wow. 

## 3. Kecepatan Index Mesin Pencari
Setelah saya memakai template ini trafik saya menjadi naik, entah bertahan berapa lama saya sendiri belum tau karena ini juga masi tester atau uji coba pada template ini sendiri. 

Tapi itu semua tergantung operator nya yang menjalan kan blog tersebut yang mana jika terus optimasi maka akan cepat pula index nya di mesin pencari. 

## 4. Unlimeted Subdomain
Karena SimPerson ini sudah menggunakan licensi maka bisa di subdomain pada blog tersebut jadi tidak perlu kuwatir akan redirect ke penyedia template tersebut. 

Mungkin untuk sementara itu dulu kenapa saya memilih dan membeli template kangrian ini, mungkin ada tambahan dari kalian langsung komentar aja gan.
